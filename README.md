# 🚨 Problem Statement  
  
Road accidents and emergency situations often suffer from delayed response because of:  
  
- lack of instant communication  
- weak network connectivity  
- panic during emergencies  
- absence of real-time location context  
- high-friction emergency systems  
- delayed guardian notifications  
  
Traditional emergency apps fail because:  
  
- voice assistants break in noisy environments  
- login systems waste critical seconds  
- emergency systems are too slow during panic situations  
- guardians lack real-time tracking visibility  
  
Rakshak AI solves this using an intelligent, offline-resilient, AI-powered emergency response ecosystem optimized for the **Golden Hour**.  
  
---  
  
# 🧠 What is Rakshak AI?  
  
Rakshak AI is an AI-powered emergency response and road safety platform acting as a digital safety ecosystem for modern users.  
  
It combines:  
  
- One-tap SOS emergency activation  
- Real-time guardian alert system  
- WhatsApp emergency notifications  
- Live location & trip sharing  
- AI safety assistant  
- Offline fallback systems  
- Smart risk analysis  
- Route tracking & recovery  
  
into a single ultra-fast mobile-first safety platform.

# ✨ Core Features  
  
## 🚨 1. Instant SOS Emergency System  
  
A massive high-contrast emergency trigger designed for real-world panic situations.  
  
### Features:
- One-tap emergency activation  
- Real-time SOS alert dispatch  
- Guardian notifications  
- Live location attachment  
- Fast response UX  
- Offline-aware architecture  
  
---  
  
## 📍 2. Guardian Live Tracking System  
  
Users can share live trips with guardians in real time.  
  
### Features:
- Real-time trip sharing  
- Continuous location updates  
- Tracking ID generation  
- Guardian monitoring dashboard  
- Emergency route visibility  
  
---  
  
## 📲 3. WhatsApp Emergency Alerts  
  
Integrated using Twilio API.  
  
### Features:
- SOS messages to guardians  
- Live location links  
- Real-time emergency alerts  
- Multi-guardian support  
- Cloud-triggered notifications  
  
---  
  
## 🤖 4. AI Safety Assistant  
  
Powered using Google Gemini AI.  
  
### Features:
- AI emergency assistance  
- Smart safety suggestions  
- Risk analysis  
- Context-aware responses  
- Emergency guidance  
  
---  
  
## 🧭 5. Last Known Path Recovery (LKPR)  
  
Rakshak continuously stores route breadcrumbs.  
  
### Features:
- Route reconstruction  
- Crash recovery assistance  
- Last active location tracking  
- Offline fallback recovery

- User sessions  
- Cloud authentication  
- Production-ready backend auth

```bash

# Enter Project Folder
cd RAKSHAK

# Install Dependencies
npm install

# Create Environment Variables
VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
VITE_GEMINI_API_KEY=your_key

# Start Frontend
npm run dev
